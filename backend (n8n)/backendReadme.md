# Backend

This backend is implemented using n8n workflows.

## Workflows

- analyze-project.json
- release-history.json
- ai-insights.json
- reports.json
- risk-center.json
- health.json
- release-details.json

## Setup

1. Import workflows into n8n.
2. Configure credentials for OpenAI, MongoDB, and webhook authentication.
3. Activate workflows.
4. Update frontend API URLs if needed.
